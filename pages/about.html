site details

The site as you see it is completely static, there is a little bit of Java Script for the animations but there are no cookies, no tracking, no ads, no pop-ups. Its developed in Jekyll on a Mac and the final built static site is uploaded to the host. The ultimate plan is to move the hosting to github pages.

Initially the we used the Simplex theme but its been heavily modified (at some point we’ll clean it up and share it). Comments are handled by Hyvor which is an ad-free, tracking free commenting service. One day comments might migrate to staticman but it’s not a priority and that would mean dealing with the database.
