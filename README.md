# TrickyBitsBlog

This is the built version of the Tricky Bits blog as seen at `https://TrickyBitsBlog.github.io`. The blog is a full static site built with Jekyll. To host this website locally simply clone and use python to serve a folder as a web sever.

```
cd TrickyBitsBlog.github.io
python3 -m http.server 4000
```

now point a web browser at `http://localhost:4000`
